# Streamlit

Streamlit 学習用リポジトリ

## 開発環境

OS : Windows OS  
Python : [3.12.10](https://www.python.org/downloads/release/python-31210/)  
pip : 25.0.1  
Streamlit : 1.50.0  
Pillow : 11.3.0
pandas : 2.3.3
matplotlib : 3.10.7

## 使用ライブラリ

・Streamlit  
　-Web アプリ開発用ライブラリ

・Pillow  
　-画像処理ライブラリ

## アプリの起動

### app.py

ルートディレクトリにいる状態で下記コマンドを実行する

```bash
streamlit run app.py
```

### main_app.py

web_app ディレクトリに移動し、下記コマンドを実行する

```bash
streamlit run main_app.py
```
